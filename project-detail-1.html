The goal of this project was to create an eCommerce website for my makeshift brand "Vahan." The project aimed to showcase my ability to integrate web development and design skills. I utilized JavaScript plugins, coded everything from scratch, designed graphics for all products on the site, and featured a website design that aligned with the brand identity of “Vahan.”

This project was an assignment for Web Scripting 1. The requirement for the project was to incorporate 5 JavaScript plugins onto a webpage.

process.

I started by organizing the project structure, setting up the necessary code files using HTML, CSS, and JavaScript. This phase ensured a clean foundation to build the rest of the project efficiently. I set up version control through Git to manage updates and changes throughout the process.

I then created a 3D logo, designed an animation in After Effects, and used Lottie.js to integrate it into the homepage. The main challenge was optimizing the animation to maintain quality while ensuring smooth performance and fast load times.

I integrated plugins like Leaflet for map functionality and Animate on Scroll for subtle animations, enhancing interactivity while maintaining site performance. I also implemented jQuery validation for the contact form to ensure smooth user experience, especially on mobile devices.

After that, I worked on the overall layout of the site, focusing on both the shop and contact pages. The shop needed to showcase products in a clean, organized way, while ensuring smooth navigation for users. For the contact page, I designed a minimal but functional layout to ensure ease of access.
